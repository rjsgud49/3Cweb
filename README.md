# 🏫 3C 인증제 조회 시스템 (3C Web)

> 교내 3C 인증제 조회를  
> **외부에서도 접근 가능하고, 더 편하게 관리할 수 있도록 개선한 웹 서비스**

---

## 📌 프로젝트 소개

기존 교내 3C 인증 시스템은  
- 내부망에서만 접근 가능  
- 복잡한 UI  
- 불편한 조회 방식  

이라는 문제를 가지고 있었습니다.

이를 해결하기 위해  
👉 **외부에서도 접속 가능한 개인화 조회 시스템**을 개발했습니다.

---

## 🚀 주요 기능

### ✅ 1. 3C 인증 정보 조회
- 개인 인증 점수 및 이수 현황 확인
- 빠르고 직관적인 UI 제공

### ✅ 2. 외부 접근 지원
- 학교 내부망 없이도 접속 가능
- PC / 모바일 환경 모두 사용 가능

### ✅ 3. 간결한 UI/UX
- 불필요한 요소 제거
- 핵심 정보 중심 화면 구성

### ✅ 4. 관리 편의성 개선
- 데이터 확인 구조 단순화
- 유지보수 용이
