🏫 3C 인증제 조회 시스템 (3C Web)

교내 3C 인증제 조회 시스템을
개인화 + 외부 접근 가능 + 관리 편의성 개선을 목표로 개발한 웹 프로젝트입니다.

기존 학교 시스템은 접근성이 제한적이고 UX가 불편했기 때문에,
이를 개선하여 언제 어디서든 쉽게 조회할 수 있는 환경을 구축했습니다.

🚀 주요 기능
✅ 1. 3C 인증 정보 조회
개인 인증 점수 및 이수 현황 확인
직관적인 UI로 빠른 정보 파악 가능
✅ 2. 외부 접근 가능
학교 내부망이 아닌 환경에서도 접근 가능
모바일 / PC 어디서든 사용 가능
✅ 3. 사용자 중심 UI/UX
복잡한 기존 시스템 → 간단한 구조로 개선
필요한 정보만 빠르게 확인 가능
✅ 4. 관리 편의성 개선
데이터 확인 및 관리 구조 단순화
유지보수 용이

💡 프로젝트 배경

기존 교내 3C 인증제 시스템은 다음과 같은 문제가 있었습니다:

❌ 외부 접근 불가 (학교 내부망 제한)
❌ 복잡한 UI
❌ 조회 과정이 번거로움

➡️ 이를 해결하기 위해
**“간편하게 조회할 수 있는 개인용 웹 서비스”**를 개발했습니다.

🎯 기대 효과
학생들의 인증 현황 확인 시간 단축
접근성 향상 (언제 어디서든 확인 가능)
사용자 경험 개선
🔧 실행 방법
npm install
npm run dev
📌 향후 개선 사항
로그인 기능 고도화
데이터 자동 동기화
통계 및 시각화 기능 추가
모바일 UI 최적화
🧠 포트폴리오용 설명
📌 프로젝트 개요

교내 3C 인증제 조회 시스템의 불편한 접근성과 낮은 사용성을 개선하기 위해
외부에서도 접근 가능한 개인화된 조회 웹 서비스를 개발했습니다.

❗ 문제 정의

기존 시스템의 문제점:

학교 내부망에서만 접근 가능
UI가 복잡하여 정보 확인이 어려움
조회 과정이 비효율적
💡 해결 방법
외부에서도 접속 가능한 웹 서비스 구축
사용자 중심 UI 설계
핵심 정보 위주의 간결한 화면 구성
🛠️ 구현 내용
React 기반 SPA 구조 설계
API를 활용한 데이터 조회 시스템 구축
컴포넌트 단위 UI 분리로 유지보수성 향상
Vite를 활용한 빠른 개발 환경 구성
🔥 핵심 성과
기존 시스템 대비 접근성 대폭 개선
불필요한 UI 제거 → 사용 시간 단축
개인 프로젝트지만 실제 서비스 형태로 구현
📊 배운 점
사용자 경험(UX)이 서비스 만족도에 미치는 영향
기존 시스템의 문제를 분석하고 개선하는 능력
프론트엔드 구조 설계 및 상태 관리 경험
🚀 확장 가능성
학사 시스템 통합 서비스로 확장 가능
통계/분석 기능 추가
모바일 앱화 가능
